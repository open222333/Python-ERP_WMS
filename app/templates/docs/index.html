登入 / 帳號

  1. 開啟後台網址 /admin/，輸入 Email 與密碼後點擊「登入」。
  2. 登入後 Token 存於瀏覽器 localStorage，有效期間內重新整理不需再次登入。
  3. 點擊側欄底部「登出」可清除 Token 並返回登入頁。
自動登入：後台登入後，直接開啟 /pos/ 或 /quick-io/ 會自動沿用同一憑證，無需重新輸入帳密。

產品分類

後台側欄 → 產品分類。用於將產品歸類，可在產品查詢時篩選。

  1. 點擊「＋ 新增分類」，輸入分類名稱。
  2. 可修改名稱或刪除（刪除前需確認無產品使用此分類）。
  3. 分類也會一同匯出 / 匯入於產品 JSON 檔案中。

產品資料

後台側欄 → 產品資料。

新增 / 編輯產品

  1. 點擊「＋ 新增產品」，填入各欄位後儲存。
  2. 點擊列表中的「編輯」可修改現有產品。
欄位說明
SKU唯一識別碼，同款商品不同規格各有獨立 SKU，可點擊「自動產生」
條碼商品條碼，可用條碼機掃描輸入，也可點擊「自動產生」
名稱 / 分類 / 單位基本資料
成本價 / 售價用於庫存金額計算與報表
最低庫存低於此值時儀表板顯示「低庫存」警示；填 0 = 不限制
最高庫存高於此值時顯示「超量」警示；填 0 = 不限制
狀態停用後不顯示於 POS 菜單品項選擇

匯出產品（JSON）

  1. 點擊「⬇ 匯出」，下載包含所有分類與產品的 JSON 檔。
  2. 檔名格式：product_YYYYMMDD_HHMMSS.json。

匯入產品（JSON）

  1. 點擊「⬆ 匯入」，選擇 JSON 檔案（格式需與匯出相同）。
  2. 系統比對 SKU：已存在的產品更新資料、不存在的新增；分類已存在則略過。
  3. 匯入完成後庫存警示自動重整。
建議先匯出現有資料作為範本，再依格式批次編輯後匯入。

倉庫管理

後台側欄 → 倉庫管理。支援多倉庫，同一產品可在不同倉庫各自存放庫存。

  1. 點擊「＋ 新增倉庫」，輸入名稱與說明後儲存。
  2. 點擊倉庫列表中的倉庫名稱，展開該倉庫的「儲位管理」。
  3. 可在倉庫下新增儲位（Location），用於更細緻的庫存分區。

庫存查詢

後台側欄 → 庫存查詢。可依倉庫或產品篩選，顯示即時庫存量。

盤點調整

  1. 點擊「盤點調整」按鈕。
  2. 選擇產品與倉庫，輸入實際盤點數量（非異動量）。
  3. 系統自動計算差異並建立「盤點調整」移動紀錄。

入庫管理

後台側欄 → 入庫管理。建立正式採購 / 進貨單，完整追蹤進貨流程。

  1. 點擊「＋ 新增入庫單」，選擇目標倉庫，填入備註。
  2. 點擊「＋ 新增品項」，選擇產品（可用條碼掃描或搜尋），輸入數量與單價。
  3. 確認無誤後點擊「確認入庫」，庫存立即增加。
狀態說明
草稿可繼續編輯品項，尚未異動庫存
已入庫庫存已增加，不可再修改
已取消作廢，不影響庫存

出庫管理

後台側欄 → 出庫管理。建立正式銷售 / 出貨單，流程與入庫單相同。

狀態說明
草稿可繼續編輯品項，尚未異動庫存
已出庫庫存已減少，不可再修改
已取消作廢，不影響庫存

庫存移動紀錄

後台側欄 → 庫存移動。顯示所有產品相關的庫存異動歷史。

移動類型觸發來源
入庫確認入庫單、快速入庫
出庫確認出庫單、快速出庫
消耗快速消耗
盤點調整庫存查詢 → 盤點調整
POS 銷售觸發的庫存異動僅系統內部記錄，不顯示於此清單（避免與正式出庫單混淆）。

快速出入庫

網址：/quick-io/，或後台側欄 → 快速出入庫（開新分頁）。

  1. 選擇倉庫與操作類型（入庫 / 出庫 / 消耗）。
  2. 點擊「＋ 加入品項」，搜尋產品並輸入數量，可加入多筆。
  3. 點擊「確認送出」完成批次異動，系統自動建立移動紀錄。
適合日常快速補貨 / 領料，不需建立正式入出庫單的情境。

菜單管理

後台側欄 → 菜單管理。管理 POS 收銀台顯示的菜單、品項與客製化選項。

建立菜單

  1. 點擊「＋ 新增菜單」，輸入菜單名稱後儲存。
  2. 點擊菜單名稱進入菜單詳細設定，包含「菜單分類」、「品項」、「選項群組」三個區塊。

菜單分類

在菜單內建立分類（如：飲料、主食、甜點），POS 點餐介面依分類顯示品項。

菜單品項

  1. 點擊「＋ 新增品項」，填入品項名稱、所屬分類、售價。
  2. 可上傳品項圖片。
  3. 勾選「連動庫存」後選擇對應產品，POS 出單時自動扣減產品庫存。
  4. 可為品項套用「選項群組」（如：冰塊、甜度）。

選項群組（客製化）

  1. 在菜單下建立選項群組（如：「甜度」），設定各選項（如：全糖、少糖、無糖）。
  2. 每個選項可附加額外費用。
  3. 品項套用選項群組後，POS 點餐時自動彈出客製化選項供顧客選擇。

菜單匯入 / 匯出

  • 匯出所有菜單：點擊「匯出全部」，下載包含所有菜單的 JSON。
  • 匯入：點擊「匯入」，選擇 JSON 檔案覆蓋現有菜單資料。
  • 單一菜單匯出：進入菜單後點擊「匯出此菜單」。

收銀台操作

網址：/pos/，適合平板或行動裝置。後台登入後自動沿用登入憑證。

點餐結帳流程

  1. 左側選擇菜單分類，點擊品項加入購物車（右側）。
  2. 點擊購物車中的品項可調整數量或刪除。
  3. 點擊「結帳」按鈕，開啟結帳視窗。
  4. 選擇付款方式；現金付款可輸入收取金額，系統自動計算找零。
  5. 可點擊折扣快捷按鈕（後台設定）或手動輸入折扣。
  6. 點擊「確認付款」完成訂單，連動庫存品項自動扣庫。

折扣說明

折扣類型說明
折扣%例如 90 = 九折（小計 × 0.9）；填 0 = 免費贈送（全額折抵）
固定折抵輸入折抵金額，例如 50 = 折抵 $50

退款

  1. 後台側欄 → 銷售記錄，找到該筆訂單。
  2. 點擊「退款」，確認後庫存回補、訂單標記為已退款。
退款後庫存自動回補，請確認實際商品已收回再操作。

銷售記錄

後台側欄 → 銷售記錄。查詢所有 POS 訂單，可依日期、狀態篩選。

狀態說明
已付款正常完成的訂單
已退款已執行退款，庫存已回補

日結報表

後台側欄 → 日結報表。查詢指定日期的收銀彙總。

  • 顯示當日總銷售金額、訂單筆數、退款金額。
  • 依付款方式分類統計金額。
  • 熱銷品項排行（依銷售數量排序）。

外送訂單

後台側欄 → 外送訂單。管理來自外送平台的訂單。

  1. 訂單由外送平台透過 Webhook 自動推送，或點擊「同步」手動拉取。
  2. 點擊訂單查看品項明細。
  3. 更新訂單狀態（備餐中 → 配送中 → 已送達 / 已取消）。

外送平台設定

後台側欄 → 平台設定。設定各外送平台的 API 金鑰與 Webhook。

  1. 選擇平台，填入 API 金鑰後儲存。
  2. 複製「Webhook 接收網址」填入外送平台後台，即可接收即時訂單推送。
  3. 設定完成後可點擊「同步菜單至平台」，將 WMS 菜單同步到外送平台。

系統設定

後台側欄 → 系統設定（需 admin 角色）。

倉庫設定

項目說明
預設倉庫開啟 POS 或快速出入庫時自動選取，仍可手動切換

POS 折扣快捷

設定結帳時可一鍵套用的折扣按鈕，顯示於 POS 結帳視窗。

欄位說明
顯示名稱按鈕上顯示的文字，例如「九折」、「員工優惠」
類型折扣%（乘法） 或 固定折抵（減法）
數值折扣% 填 90 = 九折，填 0 = 免費贈送；固定折抵填金額

POS 付款方式

自訂 POS 結帳可選擇的付款方式。

欄位說明
啟用關閉後不顯示於 POS 結帳選項
付款名稱例如：現金、信用卡、Line Pay
現金找零勾選後 POS 顯示收現欄位與找零計算（適用現金類）
排序數字越小越靠前

角色與權限

角色可操作功能
admin 全部功能，含使用者管理、系統設定、刪除資料
operator 產品、倉庫、庫存、入出庫、菜單、POS、外送；
不可管理使用者及系統設定
viewer 僅能查看資料，不可新增、修改、刪除

使用者管理

後台側欄 → 使用者管理（需 admin 角色）。

  1. 點擊「＋ 新增使用者」，填入 Email、密碼、角色後儲存。
  2. 點擊「編輯」可修改密碼或角色。
  3. 停用帳號後該使用者無法登入，資料保留。
請至少保留一個 admin 帳號，以免失去系統管理權限。

操作紀錄

後台側欄 → 操作紀錄。記錄所有登入、新增、修改、刪除等操作，可依使用者或操作類型篩選，用於稽核追蹤。
