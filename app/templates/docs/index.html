WMS 倉儲管理系統

系統使用說明文件

系統概覽

WMS 是一套整合倉儲、銷售、外送的管理系統，主要功能模組如下：

產品管理
建立產品資料、分類、匯入匯出 JSON
倉庫管理
多倉庫設定，管理儲位與庫存分佈
庫存管理
即時庫存查詢、盤點調整、批次出入庫、低庫存警示
進 / 出貨單
採購進貨、銷售出貨全流程追蹤
收銀台 POS
點餐 / 結帳、折扣、退款、菜單管理
外送管理
外送訂單接收、狀態追蹤

登入 / 帳號

  1. 開啟後台網址（見下方系統網址），自動跳轉至登入頁面。
  2. 輸入帳號（Email）與密碼，點擊「登入」。
  3. 登入後 Token 存於 localStorage，有效期間內重新整理不需再次登入。
  4. 點擊右上角「登出」可清除 Token 並返回登入頁。
自動登入 POS / 快速出入庫：後台登入後，直接開啟 POS 或快速出入庫頁面會自動沿用同一登入憑證，無需重新輸入帳密。

產品管理

後台 → 側欄「產品」分頁。

新增產品

  1. 點擊右上角「＋ 新增產品」按鈕。
  2. 填入 SKU（唯一識別碼）、名稱、分類、單位、成本價、售價。
  3. 可設定「最低庫存」與「最高庫存」，觸發庫存警示。
  4. 點擊「儲存」完成新增。

匯入產品（JSON）

  1. 點擊「⬆ 匯入」按鈕，選擇 JSON 檔案。
  2. 系統自動比對 SKU：已存在的產品更新資料、不存在的新增。
  3. 匯入完成後庫存警示自動重整。
JSON 格式：{"categories": [...], "products": [...]}。可先匯出現有資料作為範本。

匯出產品（JSON）

  1. 點擊「⬇ 匯出」按鈕，即下載包含所有分類與產品的 JSON 檔。
  2. 檔名格式：product_YYYYMMDD_HHMMSS.json。

庫存警示設定

欄位說明
最低庫存庫存低於此值時，儀表板顯示「低庫存」警示。設為 0 = 不限制。
最高庫存庫存高於此值時，顯示「超量」警示。設為 0 = 不限制。

倉庫管理

後台 → 側欄「倉庫」分頁。

  1. 點擊「＋ 新增倉庫」，輸入倉庫名稱與說明。
  2. 建立後即可在庫存、進貨單、出貨單選擇此倉庫。
  3. 支援多倉庫，同一產品可在不同倉庫各自存放庫存。

庫存管理

後台 → 側欄「庫存」分頁。

查詢庫存

可依倉庫或產品篩選，顯示即時庫存量、產品 SKU 與倉庫名稱。

盤點調整

  1. 選擇產品與倉庫，輸入實際盤點數量（非異動量）。
  2. 系統自動計算差異並建立「盤點調整」移動紀錄。

庫存移動紀錄

顯示所有產品相關的庫存異動（入庫、出庫、調撥、盤點、消耗），POS 銷售觸發的移動僅後台記錄，不列入此清單。

快速出入庫

網址：/quick-io/，也可從後台側欄「出入庫」開啟。

  1. 選擇倉庫與操作類型（入庫 / 出庫 / 消耗）。
  2. 逐筆加入產品與數量（可多筆）。
  3. 點擊「確認送出」完成批次異動，系統自動建立移動紀錄。
適合日常快速補貨 / 領料，不需建立正式進貨單的情境。

進貨單

後台 → 側欄「進貨單」分頁。

  1. 點擊「＋ 新增進貨單」，填入供應商、倉庫、備註。
  2. 加入進貨品項（產品 + 數量 + 單價）。
  3. 儲存後狀態為「草稿」；確認後點擊「確認入庫」，庫存自動增加。
狀態說明
草稿可繼續編輯，尚未異動庫存
已入庫庫存已增加，不可再修改
已取消作廢，不影響庫存

出貨單

後台 → 側欄「出貨單」分頁。流程與進貨單相同，確認後庫存自動減少。

狀態說明
草稿可繼續編輯，尚未異動庫存
已出貨庫存已減少，不可再修改
已取消作廢，不影響庫存

報表 / 分析

後台 → 側欄「儀表板」或「分析」分頁。

報表說明
庫存警示低庫存 / 超量產品清單（依 min_stock / max_stock）
庫存總覽各倉庫庫存量與庫存金額
進出貨趨勢近期進 / 出貨數量趨勢圖

系統設定

後台 → 側欄「設定」分頁（需 admin 角色）。

設定項目說明
系統名稱顯示於後台標題列
POS 折扣快捷設定結帳時可一鍵套用的折扣（%數或固定金額）；0% 代表免費贈送
菜單管理POS 菜單品項、分類、價格設定

收銀台 POS

網址：/pos/，適合平板或行動裝置操作。

點餐結帳流程

  1. 左側選擇菜單分類，點擊品項加入購物車。
  2. 可點擊購物車品項調整數量或備註。
  3. 點擊「結帳」按鈕，選擇付款方式（現金 / 信用卡 / 行動支付）。
  4. 可套用折扣快捷按鈕（後台設定）或手動輸入折扣。
  5. 確認後完成訂單，庫存自動扣除。

客製化選項

點擊品項後可設定規格（冰塊、甜度等），依後台菜單設定顯示。

退款

  1. 進入「訂單記錄」，找到該筆訂單。
  2. 點擊「退款」，確認後原品項庫存回補。
退款後庫存自動回補，請確認實際商品已收回再操作。

外送管理

後台 → 側欄「外送」分頁。用於管理外送平台訂單或自行配送。

  1. 接收訂單後顯示於外送清單。
  2. 更新訂單狀態（備餐中 → 配送中 → 已送達）。
  3. 完成後庫存相應扣除。

角色權限

角色可操作功能
admin 全部功能，包含使用者管理、系統設定、刪除資料
operator 產品、倉庫、庫存、進出貨單、POS、外送操作；不可管理使用者與系統設定
viewer 僅能查看資料，不可新增、修改、刪除

使用者管理

後台 → 側欄「使用者」分頁（需 admin 角色）。

  1. 點擊「＋ 新增使用者」，填入 Email、密碼、角色。
  2. 可修改現有使用者的密碼與角色。
  3. 停用使用者後該帳號無法登入，資料保留。
請勿刪除唯一的 admin 帳號，否則將無法管理系統。
