API 金鑰設定說明
UberEats
  1. 至 developer.uber.com 建立 App
  2. 取得 Client ID 與 Client Secret
  3. 填入 conf/config.ini 的 [UBEREATS] 段落
  4. 將 Webhook URL 貼至 Uber Developer Dashboard
foodpanda
  1. 聯繫 foodpanda 商業夥伴申請 Vendor API 存取
  2. 取得 API Key 與 Vendor Code
  3. 填入 conf/config.ini 的 [FOODPANDA] 段落
  4. 將 Webhook URL 提供給 foodpanda 技術支援
